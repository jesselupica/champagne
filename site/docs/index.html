Introduction

Champagne is a beautiful, streamlined source control UI that works with Git, Sapling, Graphite, and more.

Documentation is actively being written. In the meantime, check out the GitHub repository for setup instructions and source code.

What is Champagne?

Champagne provides a graphical interface for version control, built for an opinionated, agent-first development cycle. It supports multiple VCS backends through a pluggable driver architecture.

Supported Backends

  • Git — raw git commands
  • Sapling — reference implementation
  • Graphite — Git wrapper for stacked PRs
  • Git Branchless — Git extension

Quick Install

Install the VS Code extension from the marketplace:

$ ext install jesselupica.champagne-scm
