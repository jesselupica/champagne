# Champagne

A standalone fork of Interactive Smartlog (ISL) from the Sapling SCM project.

## Quick Start

```bash
# Install dependencies
yarn install

# Run web app in development mode
yarn dev browser --launch .

# Run VSCode extension in development mode
yarn dev vscode --launch .
```

## What's Included

- **ISL Web App**: React-based UI for version control visualization
- **ISL Server**: Node.js backend for repository operations
- **VSCode Extension**: Full IDE integration
- **Shared Libraries**: Reusable components and utilities
